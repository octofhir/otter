# Task 97 Expansion Audit

Date: 2026-05-06

Command source:

- `cargo expand -p otter-macros --test js_namespace`
- `cargo expand -p otter-macros --test js_class`
- `cargo expand -p otter-macros --test raft`

The representative expansions were checked for the Task 97 generated-shape
contract:

- generated data is `static` `NamespaceSpec` / `ClassSpec` data;
- every generated callable uses `NativeCall::Static(path)`;
- no runtime registry, metadata parsing, boxed closure, or bootstrap mutation
  is generated by the macros;
- installation remains explicit through Task 96 builders/bootstrap.
